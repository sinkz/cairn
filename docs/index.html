Quick install

Instalação rápida

Install the CLI first. Create the vault where you want it.

Instale a CLI primeiro. Crie o vault onde você quiser.

Release installers download standalone binaries from GitHub Releases, verify checksums, and place cairn on your PATH. Python is not required for this install path.

Os instaladores baixam binários standalone do GitHub Releases, validam checksums e colocam cairn no PATH. Python não é necessário neste caminho de instalação.

Linux / macOS curl -fsSL https://sinkz.github.io/cairn/install.sh | sh
Windows PowerShell irm https://sinkz.github.io/cairn/install.ps1 | iex
Create a vault Criar um vault cairn init --path PATH_TO_VAULT --profile personal

Workflow

Fluxo

The basic loop is search, retrieve, solve, and write back.

O fluxo básico é buscar, recuperar, resolver e registrar.

Notes can represent incidents, access processes, support procedures, library references, and decisions. Agents use the vault as working context instead of opening everything.

Notas podem representar incidentes, processos de acesso, procedimentos de suporte, referências de bibliotecas e decisões. Agentes usam o vault como contexto de trabalho sem abrir tudo.

01

Search

Buscar

Find likely notes before opening full files.

Encontre notas prováveis antes de abrir arquivos completos.

02

Retrieve

Recuperar

Return a budgeted packet or only the matching section.

Retorne um pacote com orçamento ou só a seção relevante.

03

Solve

Resolver

Use old decisions and fixes as live working context.

Use decisões e correções antigas como contexto de trabalho.

04

Write back

Registrar

Update the closest note or capture new reusable knowledge.

Atualize a nota mais próxima ou registre novo conhecimento.
